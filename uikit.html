Custom Utility Classes

Utility Classes for background color

bg-muted
set background to muted color of black (similar to wolf gray color)
bg-blue
set background color to FTA blue
bg-orange
set background color to FTA orange
bg-orange-light
set background color to FTA orange light version (used mostly on buttons)
bg-white
set background color to white
bg-black
set background color to FTA black

Custom Typography Classes

text-blue

(set text color to FTA blue)

text-orange

(set text color to FTA orange)

muted

(set text color to FTA muted)

text-white

(set text color to FTA white)

p-bold

(set text to bold)

hover-orange

(set hover color to light orange)

text-orange hover-orange

(default hyperlink with orange text color and light orange hover color)

Shadow and Border Classes

no-shadow

remove container/card/button shadow effect

drop-shadow-sm

add a small drop shadow (for small cards or buttons)

drop-shadow-md

add a medium size drop shadow (for medium size cards with muted or darker container background)

drop-shadow-lg

add a large drop shadow (for large cards with muted or darker container background)

no-border

removes border (in this example it removes the header bottom border)

Note: All Cards has a default subtle drop shadow (the "no-border" example shows default drop shadow)
